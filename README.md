# ext4cats' site

Someone's special corner in the world wide web.

## Building

You will need [Node.js](https://nodejs.org) and [pnpm](https://pnpm.io) to build this project.

```bash
# install dependencies
pnpm install

# run development server
pnpm dev

# or do a production build
pnpm build
```
